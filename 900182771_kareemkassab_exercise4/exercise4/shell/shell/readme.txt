Hello there,
this program takes a command and executes it in an infinite loop of commands unti you command "exit"
------------------------------------------------------------------------
there is a make file which contains Rules to generate:
executable file "SIMPLE" using object codes[prerequisites]
object codes using assembly codes[prerequisite]
assembly codes using preprocessed codes [prerequisite]
preprocessed codes using source codes [prerequisite]

the code is in "main" with suitable file kind extension
-----------------------------------------------------------------------------------------------------
to build the program, simply open its directory in the terminal and make
then execute through "./SIMPLE"
-------------------------------------------------------------
to run the program, execute through ./SIMPLE
enter a command after the other, when done, press exit and it will break the loop
