Hello there,
this program prints rectangular and square shapes made of "*" with the sizes you desire
------------------------------------------------------------------------
there is a make file which contains Rules to generate:
executable file "DRAW" using object codes[prerequisites]
object codes using assembly codes[prerequisite]
assembly codes using preprocessed codes [prerequisite]
preprocessed codes using source codes [prerequisite]

the code is in "main" with suitable file kind extension
-----------------------------------------------------------------------------------------------------

to run the program, execute through ./DRAW
u enter the required shape through 1 for square or 2 for rectangle, then enter the lenght or length and width as needed, then it displays the shape as *'s.
