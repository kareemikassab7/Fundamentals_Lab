Hello there,
this program takes the size of a queue from the user and lets the user fill the queue, then it takes the number of elements needed to be swapped from the beginning and swaps them.
---------------------------------------------------------------------
there is a make file which contains Rules to generate:
executable file "SWPQ" using object codes[prerequisites]
object codes using source codes[prerequisite]

the code is in "SWPQ.cpp" with suitable file kind extension
-----------------------------------------------------------------------------------------------------
to build the program: simple enter the directoryu on the terminal and type "make"
it will produce SWPQ
----------------------------------------------------------
to run the program, execute through ./SWPQ
enter the size then fill the queue; after that,enter the number of elements you want to be swapped.
results will be automatically displayed on the screen
