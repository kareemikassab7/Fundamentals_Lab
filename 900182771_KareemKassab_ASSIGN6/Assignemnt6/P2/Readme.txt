this program should read the points from the file  where there are objects including(square, circle, rectangle, triangle, house, and RecCircle)

------------------------------------------------------------------------
there is a make file which contains Rules to generate:
executable file "MAPOBJ" using object codes[prerequisites]
object codes using source file "Source.cpp"
the code is in "main" with suitable file kind extension
-----------------------------------------------------------------------------------------------------
to build the program, simply open its directory in the terminal and make
then execute through "./MAPOBJ"
-------------------------------------------------------------
to run the program, execute through ./MAPOBJ
all the data will be displayed automatically on the screen
